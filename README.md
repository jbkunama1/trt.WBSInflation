## 📚 Projektübersicht | Project Overview

Dieses Projekt bietet eine **interaktive Lernplattform** zum Verständnis von Inflation im Kontext des WBS-Lehrplans 2016 für Baden-Württemberg. Das Projekt kombiniert:

🎯 **Theoretische Grundlagen** • 📈 **Infografiken & Datenvisualisierung** • 🧠 **Interaktives Quiz** • 💡 **Live-Dashboard**

---

## 🎯 Projektziele | Learning Objectives

- ✅ Verständnis für das Konzept der Inflation entwickeln
- ✅ Ursachen und Folgen von Inflation verstehen
- ✅ Geldpolitik und ihre Auswirkungen erkennen
- ✅ Kaufkraft und reale Zinssätze analysieren
- ✅ Praktische Beispiele aus der Realwirtschaft
- ✅ Befähigung zu kritischem wirtschaftlichem Denken

### 3️⃣ 🎮 Interaktives Quiz | Interactive Quiz System

Das **Quiz-System** ist das interaktive Kernstück mit:

#### 🎯 Quiz-Features:
- ✅ **Umfangreiche Fragendatenbank**: 50+ Fragen
- 📱 **Responsive Design**: Funktioniert auf allen Geräten
- 📊 **Automatische Auswertung**: Sofortiges Feedback
- 👤 **Benutzerverwaltung**: Admin-Panel für Dozenten
- 📈 **Ergebnisspeicherung**: CSV-Export der Ergebnisse

#### ❓ Beispiel-Fragen aus dem Fragenpool:

| Frage | Antwort | Schwierigkeit |
|-------|--------|---------------|
| Was ist ein typisches Anzeichen für Inflation? | Steigende Preise | ⭐ Leicht |
| Was versteht man unter Kaufkraft? | Was man für Geld kaufen kann | ⭐ Leicht |
| Wer steuert die Geldpolitik im Euroraum? | Europäische Zentralbank (EZB) | ⭐⭐ Mittel |
| Welche Maßnahme hilft gegen Inflation? | Zinserhöhung | ⭐⭐ Mittel |
| Welche Gruppen spüren Inflation besonders stark? | Menschen mit wenig Geld | ⭐⭐ Mittel |

#### 📋 Quiz-Struktur:
```
5 Seiten (Lektionen)
├── Start-Seite
├── Seite 1: Inflationsgrundlagen
├── Seite 2: Ursachen & Folgen
├── Seite 3: Geldpolitik
├── Seite 4: Fallstudien
├── Seite 5: Zusammenfassung
└── Ergebnisseite
```

**Zugriff:** 
- `quiz/index.php` - Hauptseite
- `quiz/start.php` - Quiz starten
- `quiz/admin.php` - Admin-Panel

## 🚀 Verwendung | How to Use

### 📚 Für Schüler | For Students:

1. **Essay lesen**: Öffne `essay/01_Essay_Inflation_WBS_1.0.pdf` für theoretisches Verständnis
2. **Infografiken studieren**: Schaue dir die Grafiken in `essay/Grafiken/` an
3. **Dashboard erkunden**: Öffne `dashboard/index.html` für visuelle Daten
4. **Quiz absolvieren**: Besuche `quiz/index.php` und teste dein Wissen
5. **Ergebnisse erhalten**: Sofortiges Feedback nach dem Quiz

### 👨‍🏫 Für Lehrkräfte | For Educators:

1. **Admin-Panel**: Zugriff auf `quiz/admin.php`
2. **Ergebnisse anschauen**: CSV-Dateien in `quiz/data/`
3. **Materialien anpassen**: Bearbeite `quiz/fragenpool_komplett.json`
4. **Dashboard im Unterricht zeigen**: Projiziere `dashboard/index.html`

## 💾 Installation & Betrieb | Installation & Running

### 📋 Voraussetzungen | Requirements:
```
- Web Server (Apache, Nginx)
- PHP 7.0+
- Browser mit JavaScript-Unterstützung
```

### 🚀 Schritt-für-Schritt | Step-by-Step:

1. **Repository klonen:**
   ```bash
   git clone https://github.com/jbkunama1/trt.WBSInflation.git
   ```

2. **Zu Verzeichnis navigieren:**
   ```bash
   cd trt.WBSInflation
   ```

3. **Web-Server starten** (lokale Option):
   ```bash
   # Mit PHP built-in server
   php -S localhost:8000
   
   # Oder mit Apache/Nginx
   ```

4. **Im Browser öffnen:**
   - **Quiz**: http://localhost:8000/quiz/
   - **Dashboard**: http://localhost:8000/dashboard/
   - **Essay**: essay/01_Essay_Inflation_WBS_1.0.pdf
